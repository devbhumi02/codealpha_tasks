# Calculator Web Application

A modern multi-theme calculator built using Vanilla JavaScript as part of the CodeAlpha Frontend Development Internship.

The application performs real-time arithmetic calculations and maintains a calculation history while providing an intuitive user interface and keyboard support.

## Key Features

* Perform arithmetic operations (+, −, ×, ÷, %)
* Real-time expression evaluation
* Calculation history panel
* Backspace & clear controls
* Keyboard input support
* Light / Dark / Black theme switcher
* Responsive layout for all screen sizes

## Tech Stack

HTML5
CSS3 (Flexbox & Grid)
JavaScript (DOM Manipulation & Events)


## How to Run

Simply open the `index.html` file in any modern browser.

No dependencies or installations required.

## Project Objective

To understand DOM handling, event listeners, dynamic UI updates and state management using pure JavaScript without external libraries.
