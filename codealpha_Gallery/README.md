# Filterable Image Gallery

A responsive image gallery application that allows users to browse and filter images by category with an interactive lightbox preview.

Developed during the CodeAlpha Frontend Internship to demonstrate DOM filtering logic and UI interaction handling.

## Key Features

* Category based filtering
* Lightbox full screen preview
* Next / Previous navigation controls
* Smooth hover animations
* Lazy image loading
* Responsive grid layout

## Categories Included

Nature
Technology
Animals
Space
Food

## Tech Stack

HTML5
CSS3 (Grid Layout & Animations)
JavaScript (Event Handling & Dynamic Filtering)

## Folder Structure

Images are organized inside the `galleryimg` directory and dynamically filtered using class selectors.

## How to Run

Open `index.html` in a browser.

## Project Objective

To implement DOM filtering techniques, event delegation and modal UI behaviour without frameworks.
