# Web Music Player

A browser-based music player inspired by modern streaming platforms.
Built using JavaScript Audio API to manage playback, progress tracking and playlist interaction.

## Key Features

* Play / Pause audio
* Next / Previous track navigation
* Progress bar seeking
* Volume control
* Song duration display
* Dynamic playlist cards
* Auto-play next track

## Tech Stack

HTML5
CSS3 (Flexbox Layout)
JavaScript (Audio API & DOM Manipulation)

## Folder Structure

`songs/` → audio tracks
`images/` → cover artwork

## How to Run

Open `index.html` in any browser that supports HTML5 Audio.

## Project Objective

To learn handling of media elements, real-time UI synchronization and user interaction with audio streams using pure JavaScript.

## Future Scope

* Shuffle & repeat modes
* Playlist management
* Local storage support
* Mobile optimized player UI
